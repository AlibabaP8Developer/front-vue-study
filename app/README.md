# app

> 目录介绍

src/assets: 一般是静态资源
src/components：非路由组件（全局组件）
App.vue: 唯一的根组件
main.js: 程序入口文件，也是整个程序中最先执行的文件
babel.config.js: 配置文件

## Project setup
```
npm install
```
